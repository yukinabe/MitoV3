# Mito V3

Third iPhone-first SwiftUI prototype for Mito, a study timer where completed
focus sessions become visible pixel RPG progress.

## Baseline

This prototype starts from the latest `MitoV2` UI branch and keeps its layered
pixel-art screens, local progression state, flashcards, battles, and team
collection. V3 is the isolated branch for tightening the core product loop:

1. Start a focus session quickly.
2. Complete focused study time.
3. Earn ATP and gold.
4. Spend earned progress in the RPG layer.

## Open The App

Open `MitoV3.xcodeproj` in Xcode and run the `MitoV3` scheme on an iPhone
simulator.
