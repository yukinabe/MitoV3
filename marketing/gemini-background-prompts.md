# Mito — Subtle Background Image Prompts (for body slides 2+)

Atmospheric, understated backgrounds for the non-hook slides so every slide is full-bleed
(not boring cream). Loosely connected to study/student life — vibe, not literal. Reusable: the
same background can repeat across posts; just rotate them so it's not identical every slide.

## How to use
1. Paste the **style prefix** + a scene.
2. Ask for **4:5 vertical (1080×1350)**.
3. **DARK** backgrounds → white text. **LIGHT** backgrounds → dark text. (I add a subtle gradient so text always reads.)
4. Send them back; name them `bg-dark-#` / `bg-light-#` or just send and I'll sort.

## Style prefix (prepend to every scene)
> Moody, cinematic, slightly out-of-focus background photo, muted desaturated tones, soft natural light, lots of calm empty/even space for a text overlay, shot on an iPhone, subtle film grain. No faces, no text, no logos, nothing sharp competing for attention. Vertical 4:5. This is a quiet background, not the subject.

## Tips
- **Understated wins.** If it looks like a "nice photo," it's too busy for a background. Blurrier/emptier = better.
- Generate 2 variations each; keep the calmest one.
- A few **dark** + a few **light** is all you need — they rotate across every carousel.
- No text/logos in the image — headlines go on in Figma.
- Optional: a couple of **texture** shots (blurred handwritten notes, crumpled paper, a highlighter swipe) make great backgrounds for "the problem" slides.
